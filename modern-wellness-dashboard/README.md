# Modern Wellness Dashboard

A clean, minimalist application for tracking wellness metrics with a focus on aesthetics and usability.

## Features

- **Modern UI Design**: Clean, minimalist interface with thoughtful typography and spacing
- **Wellness Tracking**: Monitor water intake, steps, sleep, and mood
- **Mindfulness Exercises**: Built-in breathing exercises for stress relief
- **Activity Logging**: Easy logging of various wellness activities
- **Progress Visualization**: Circular progress indicators for quick status assessment

## Installation

1. Clone or download this repository
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python main.py
   ```

## Usage

- **Home Screen**: View your daily wellness metrics and progress
- **Mindful Button**: Start a guided breathing exercise
- **Log Activity**: Track various wellness activities
- Navigate between screens using the interface controls

## Design Philosophy

This application follows modern design principles:

- **Minimalism**: Clean interface with only essential elements
- **Consistency**: Uniform spacing, typography, and color scheme
- **Visual Hierarchy**: Clear organization of information
- **Subtle Animations**: Gentle movements to enhance user experience
- **Thoughtful Color Palette**: Calming colors that promote wellness

## Technologies Used

- **Kivy**: Cross-platform Python framework for UI development
- **Python**: Backend logic and data handling

## Customization

You can easily customize the application by modifying:

- Color scheme in the constants at the top of main.py
- Wellness metrics displayed on the home screen
- Breathing exercise parameters in the BreathingExercise class
- Activity types in the LogScreen class
