# 📅 Kivy Events Reminder App — Spec + Step-by-Step Roadmap

## 1. Overview

Приложение на Python (Kivy), которое:
- Хранит важные даты (события)
- Напоминает о них
- Вычисляет годовщины
- Сортирует по месяцам
- Позволяет фильтровать события по месяцу

---

## 2. Core Features

- Добавление события (название + дата)
- Удаление события
- Список событий
- Сортировка по месяцам
- Фильтр по месяцу
- Подсчёт годовщины
- Уведомления (внутри приложения)

---

## 3. Data Model

### Event
```
id: int
name: str
date: YYYY-MM-DD
```

---

## 4. Storage

Варианты:
- JSON (на старте)
- SQLite (расширение)

Файл:
```
data/events.json
```

---

## 5. UI Structure (Kivy)

### Screens:
- Main Screen (список событий)
- Add Event Screen
- Month Filter Screen

---

## 6. Components

- Event Card
- Add Button
- Month Selector
- Notification Label

---
